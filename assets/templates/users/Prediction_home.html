Select how you'd like to provide your physiological sensor data. Both methods use our two-stage ML pipeline for accurate affective state classification.

CSV File Upload

Upload a pre-recorded sensor CSV file exported from your wearable device. Features are automatically extracted across the entire recording.

  • Wrist signals: W_EDA, W_TEMP, W_BVP, W_ACCx/y/z
  • Chest signals: C_ECG, C_RESP, C_EMG, C_EDA, C_TEMP, C_ACCx/y/z
  • Adjustable Stage-A stress threshold
  • Best for real recorded sensor exports

Manual Value Entry

Enter a single scalar value per signal channel. StressWise generates a synthetic 60-second constant window and extracts features automatically.

  • Enter 14 signal values manually
  • Synthetic 240-sample window (60s × 4Hz)
  • Good for quick spot-checks and demos
  • Adjustable Stage-A stress threshold

Stage A — Stress Detection

A binary classifier determines whether your current physiological state is stressed. Confidence must exceed the configured threshold (default 35%).

02
Stage B — State Refinement

If Stage A predicts non-stress, a second 3-class model identifies whether you are in Baseline, Amusement, or Meditation state.

03
Result — Label + Confidence

You receive a final affective state label (Baseline, Stress, Amusement or Meditation) together with a percentage confidence score.
